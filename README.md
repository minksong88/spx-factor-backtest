# SPX 팩터 백테스트 — 추정치·펀더멘탈·모멘텀 기반 종목선택

> 목적: SPX 구성종목 대상으로 (1) **어떤 추정치/펀더멘탈 지표가 금리에 민감한지** 찾고,
> (2) 매크로를 배제한 채 **추정치·펀더멘탈·모멘텀만으로 롱숏 종목선택 로직**을 발굴.
> 기존 `macro-reaction-backtest`(자산군 국면반응)와 별개 프로젝트 — 여긴 **종목단위 크로스섹셔널**.

### ⚠ 백테스트 무결성 원칙
- **look-ahead 방지:** 추정치·가격은 일별 시점데이터라 그대로 사용. 트레일링 펀더멘탈은 `fiscal_date`(분기말)가 아니라 **리포트 공개 이후**에만 알 수 있으므로 `report_lag_days`(기본 60일) 시프트 후 ffill (`data.fundamentals_asof`).
- **생존편향 방지:** 매 리밸런스 시점의 `pit_universe(as_of)`로 그 시점 실제 SPX 구성종목만 사용.
- **멀티플 일별화:** 필요 시 분기 `per`에서 EPS를 역산해 일별 가격에 적용(선택). 포워드 P/E는 `BEST_PE_RATIO`로 이미 일별.

## 1. 연구 A — 금리민감 지표 찾기

**질문:** SPX 종목들의 추정치·펀더멘탈 지표 중, 10Y 금리 변화에 민감하게 반응하는 게 무엇인가?

접근 후보 (설계 확정 전):
- **A1. 팩터수익의 금리민감도:** 각 지표로 크로스섹셔널 롱숏 팩터(상위분위−하위분위) 구성 → 그 팩터수익률의 10Y 변화 대비 β. "어떤 스타일이 금리국면에서 작동하나".
- **A2. 추정치 리비전의 금리반응:** `BEST_EPS` 상향/하향(리비전)이 금리국면(상승/하락, 돌파/레인지)별로 달라지는지. 기존 repo의 국면분할 로직 재사용 가능.
- **A3. 종목 금리베타 ↔ 펀더멘탈:** 종목별 수익률의 금리베타를 구하고, 그 베타가 어떤 펀더멘탈(듀레이션 성격=고멀티플·저수익·고성장)로 설명되는지 회귀.

## 2. 연구 B — 매크로 배제, 종목선택 롱숏 로직

**질문:** 추정치 + 펀더멘탈 + 모멘텀만으로 롱숏 종목을 어떻게 뽑아야 성과가 좋은가?

**초기 가설 (사용자):**
- **숏 후보:** RSI 과매수 + 멀티플 높음 + 펀더멘탈 나쁨(악화).
- **롱 후보:** 그 반대 (과매도 + 싸고 + 펀더멘탈 양호).
- **모멘텀 크래시 가드:** 모멘텀 최상위 종목은 숏에서 제외 (강한 추세는 과매수여도 계속 감).

구현 방향:
- 팩터 3축 z-score 합성 — **밸류**(per·pbr·ev_ebitda·BEST_PE 저평가), **퀄리티/추정치**(margin·roic 양호 & BEST_EPS 상향), **역과매수**(RSI 높으면 숏 가점) — 를 결합해 랭킹.
- 모멘텀(예: 12-1M) 상위 N% 는 숏 유니버스에서 배제(가드).
- 월간 리밸런스 · 섹터중립 옵션 · 분위(quintile) 롱숏 · 수익곡선/샤프/MDD/턴오버 산출.
- 각 팩터 단독 → 결합 순으로 기여도 분해.
