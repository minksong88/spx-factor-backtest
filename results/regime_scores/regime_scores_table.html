실질금리(TIPS)×기대인플레(BEI) 4사분면 국면에서 각 ETF의 반응 스코어 β. β = 실질금리 1%p 움직일 때 자산의 % 반응 = mean(블록수익 / |Δ실질|). 전체표본 44블록 · 데이터 2026-07-22까지 · 유니버스 64종.

  • β 부호 = 방향. 양(+) = 롱 후보, 음(−) = 숏 후보. 컬럼 헤더 클릭 → 오름/내림 정렬.
  • β 크기 비교는 같은 국면(탭) 안에서만. 국면끼리 β 절대값 비교 금지(막대는 그 국면 최대치 기준 스케일).
  • 블록수 작으면(≤7) 신뢰도 낮음 — 티커 옆 ⁿ 표시. 직전수익(리버설/모멘텀) 필터는 붙이지 말 것.
