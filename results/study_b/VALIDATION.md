# 연구 B — 파이프라인 검증 (분기, 3축 합성)

> 목적: 코드 본격화 전에 **데이터·신호가 실재하는지** 실DB(Supabase)로 검증.
> 방식: PIT SPX · 분기 리밸런스 · 크로스섹셔널 z-score · 분위(5) 롱숏 · 포워드 1분기 수익.
> 원자료: [`validation_quarterly.csv`](validation_quarterly.csv) (SQL로 실측, 2021-12~2026-06, 19분기).

## 이번 검증에 넣은 것 (핵심 3축만)
- **밸류**: `−z(BEST_PE_RATIO)` (포워드 P/E 낮을수록 롱)
- **추정치 리비전**: `z(BEST_EPS 3M 변화율)` (상향일수록 롱)
- **역과매수**: `−z(RSI14)` (과매도일수록 롱)
- **모멘텀 가드**: 12-1M 모멘텀 상위 10%는 숏북에서 제외
- 스코어 = `−z(fwd_pe) + z(rev) − z(rsi)`  (님 초기 가설)
- ※ 퀄리티축(roic·마진·레버리지)은 분기 asof라 이번 검증엔 미포함 → 전체 python 런에서 합류.

## 결과
| 지표 | 값 |
|---|---|
| 평균 분기 LS | **+0.80%/분기** (승 12 / 패 7) |
| **평균 IC** | **≈ 0.003 (사실상 0)**, IC 범위 +0.145 ~ −0.213 (극심한 국면의존) |
| 최악 분기 | 2025-06 −9.0%(IC −0.21), 2024-09 −6.3%(IC −0.17) |
| 최고 분기 | 2021-12 +6.6%, 2022-12 +6.2%, 2025-03 +6.1% |

## 해석
1. **파이프라인 무결성 확인**: 생존편향 없는 PIT 유니버스(19분기 각 ~500종), look-ahead 없는 시점데이터, 리포트 지연 반영. 전 분기 정상 산출.
2. **님 초기 가설 = 밸류+단기리버설 팩터**. 2021~2026에서 **평균 IC ≈ 0**, 국면의존 극심.
3. **모멘텀 멜트업에서 붕괴**: 2024 하반기·2025 상반기 AI랠리 구간에서 "과매수 숏 / 고멀티플 숏"이 정면으로 틀림 → LS −6~−9%. 리버설 장(2021-12, 2022-12, 2025-03)에선 +6% 벌이. 즉 **알파가 아니라 스타일 베타**에 가까움.
4. → 리서치 방향: (a) 퀄리티축 합류, (b) RSI 부호/역할 재검토(이 장에선 모멘텀 지속이 우세), (c) 국면조건부(모멘텀 인지형) 결합, (d) IC 안정적인 축이 무엇인지 팩터 단독 분해.

## 데이터 품질 메모
- 일부 분기말 날짜가 소수 티커의 오염된 날짜(주말 등)로 `max(date)`에 잡혀 정확일치 가격조인이 전 종목 탈락시킴 → **밀집 거래일(≥400종) 캘린더 + as-of 조인**으로 해결. `study_b_pick.py`에도 동일 수정 반영.
